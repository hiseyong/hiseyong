## About me

I'm an Electrical and Computer Engineering student at **SUNY Korea (Stony Brook University)**, interested in **Embedded Systems**, **Communications**, and **Physical AI**.

I work across robotics, autonomous systems, and signal processing — from sim-to-real RL for quadrupeds and aircraft, to physics-constrained deep learning for modulation classification.
